## 🧠 Como eu penso tecnologia

<p align="center">
  <img src="assets/systems-card.svg" alt="Systems thinking animated card" width="70%" />
</p>

Tecnologia, para mim, não é só código.

É uma forma de enxergar problemas, reduzir ambiguidade, entender dependências e tomar decisões melhores.

Gosto de olhar para sistemas perguntando:

- onde o fluxo trava?
- onde a decisão demora?
- onde a dependência se acumula?
- onde o retrabalho nasce?
- onde a métrica está contando uma história que ainda não foi verbalizada?

Esse olhar é o que conecta minha trajetória técnica com minha atuação em gestão.
